This is a very simple Node.js server hosting static hmtl pages.

This project covers the bascis of creating a simple web server using only the built-in Node modules,Express was not used in this project. Modules used for this project are http,fs and path modules where all of them together create a small web server that is hosting only small static pages.

# Changes to the project

Due to the different parts of TheOdin Project curriculum,Express framework was installed for this project so the end result looks different from the beginning. The changes to this project can be seen and tracked in the different commits that have been done during the work of configuring the web server.
